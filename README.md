## Intructions to run the projects
- **Client side:**
```
cd app
yarn start
```
- **Server side:**
```
cd server
yarn start
```
- **Crypto side:**
```
cd crypto
npx hardhat compile
npx hardhat test
npx hardhat run scripts/nft-script.js
npx hardhat node
npx hardhat run scripts/portal-script.js --network localhost
```
